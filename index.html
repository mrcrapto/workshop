04 — foundations

المفاهيم الأساسية

قبل أي شيء، هذه هي المصطلحات الأربعة التي ستسمعها في كل محادثة عن الذكاء الاصطناعي.

LLMs النماذج اللغوية الكبيرة

نماذج ذكاء اصطناعي درّبت على كميات هائلة من البيانات، تتنبأ بالكلمة التالية. هي محرّك الفهم والكتابة في كل أداة AI تستخدمها.

Prompt الموجِّه

السؤال أو التعليمات التي تكتبها للنموذج. جودة الموجّه تحدد جودة الإجابة — السياق، الأمثلة، والقيود تصنع الفارق الكامل.

Tokens الرموز

الوحدات الصغيرة التي يقرأها النموذج — قد تكون كلمة، جزء كلمة، أو علامة ترقيم. كل نموذج له حدّ أقصى للرموز، وكل رمز له تكلفة.

API واجهة البرمجة

جسر يربط برامجك بنماذج AI مباشرة، بلا واجهة محادثة. من خلاله تبني تطبيقاتك وأدواتك الخاصة وتُدخل الذكاء في أي منتج.

skill 02 / 06 — model council

Model Council

AK
Andrej Karpathy
@karpathy
As a fun Saturday vibe code project, I hacked up an **llm-council** web app. It looks exactly like ChatGPT except each user query is 1) dispatched to multiple models on your council using OpenRouter:
"openai/gpt-5.1",
"google/gemini-3-pro-preview",
"anthropic/claude-sonnet-4.5",
"x-ai/grok-4",
2) all models see each other's anonymized responses and rank them, then 3) a "Chairman LLM" takes all that as context and produces the final response.

Quite often, models are surprisingly willing to select another LLM's response as superior to their own — making this an interesting evaluation strategy.

skill 06 / 06 — when to use ai

متى نستعمل الـAI

المهارة الأهم: أن تعرف متى تطفئ الذكاء الاصطناعي. أحيانًا الإنسان أبلغ من الآلة، وأحيانًا الأداة البسيطة أصدق.

لا تستعمل AI عندما:

01المعلومات الحساسة او العاطفية
02المهمة بسيطة جدًا، الكتابة بنفسك أسرع من صياغة الموجّه.
03تحتاج دقة 100% في أرقام أو حقائق حساسة والتحقق ليس متاحًا.
04فقدان حس الابداع مع الأستخدام المتكرر
05كتابة الرويات و النصوص و السيناريوهات

استعمل AI عندما:

01المهمة متكررة، تستهلك وقتًا، وتحتاج اتساقًا (تلخيص، صياغة، تنسيق).
02تحتاج عصف ذهني — توليد 30 فكرة في دقيقتين بدل ساعتين.
03تواجه ورقة بيضاء — اطلب مسودة أولى لتنطلق منها.
04تحتاج ترجمة، شرح بأسلوب مختلف، أو تبسيط مفهوم معقّد.
05تبني نموذجًا أوليًا (prototype) — السرعة أهم من الكمال.
